## Restaurant Chatbot
A WhatsApp-style chatbot for a restaurant named "Spice Garden", built with HTML, CSS, and JavaScript. Integrated with the Claude AI API for natural language processing.

### Features
- Realistic WhatsApp-like chat interface with typing indicators and message bubbles
- Handles customer inquiries related to the restaurant's menu, timings, reservations, and delivery
- Provides personalized responses based on the user's input
- Integrates with the Claude AI API to generate human-like responses
- Displays quick reply buttons for easy navigation
- Responsive design that adapts to different screen sizes

### Technologies Used
- HTML5, CSS3, and JavaScript for the frontend
- Claude AI API for natural language processing and response generation
- Netlify for hosting and deployment

3. Start chatting with the restaurant chatbot!

### Customization
- Modify the `RESTAURANT_INFO` object in the JavaScript code to update the restaurant details, menu items, and pricing.
- Customize the styling by modifying the CSS code in the `<style>` section of the HTML file.
- Replace the placeholder images with your own restaurant-specific images.
